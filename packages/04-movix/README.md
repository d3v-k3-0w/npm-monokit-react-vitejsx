# Movix

Movix es una aplicación de referencia para avances y carteleras de películas construida con React, Redux Toolkit y Axios.

## Dependencias

- `axios`: Un cliente HTTP basado en promesas para el navegador y Node.js, utilizado para realizar solicitudes a APIs.
- `react` y `react-dom`: Bibliotecas de JavaScript para construir interfaces de usuario.
- `@reduxjs/toolkit` y `react-redux`: Redux es una biblioteca de JavaScript para gestionar el estado de la aplicación.
  Redux Toolkit es la versión oficial y opinionada para construir aplicaciones Redux. React Redux es la biblioteca oficial de enlaces de React para Redux.
- `react-router-dom`: Una colección de componentes de navegación para React.
- `dayjs`: Una biblioteca de JavaScript minimalista para manipular fechas y horas.
- `react-circular-progressbar`: Un componente de React para crear y mostrar barras de progreso circulares.
- `react-icons`: Una biblioteca de iconos para React que soporta una amplia variedad de conjuntos de iconos.
- `react-infinite-scroll-component`: Un componente de React para implementar la funcionalidad de scroll infinito.
- `react-lazy-load-image-component`: Un componente de React para cargar imágenes de manera perezosa.
- `react-player`: Un componente de React para reproducir una variedad de URLs de video, incluyendo archivos, YouTube, Facebook, Twitch,
  SoundCloud, Streamable, Vimeo, Wistia, Mixcloud, y DailyMotion.
- `react-select`: Un componente select controlado flexible para React, diseñado para reemplazar el select nativo.
- `sass`: Un preprocesador de CSS que permite usar características que no existen en CSS todavía como variables, anidación, mixins, herencia y otros.
- `vite`: Un compilador de front-end que proporciona un desarrollo más rápido y un menor tiempo de construcción.
- `eslint`: Una herramienta de linting de código abierto que ayuda a identificar y reportar patrones encontrados en el código de JavaScript,
  lo que hace que el código sea más consistente y evita errores de sintaxis y de lógica.
- `@types/react` y `@types/react-dom`: Proporcionan las definiciones de TypeScript para las bibliotecas de React y ReactDOM.
